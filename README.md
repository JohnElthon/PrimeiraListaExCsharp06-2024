# Primeira Lista de Exercícios - C#

Exercícios de lógica de programação feitos durante o curso no Senac Santos em 2024, usando C# no Visual Studio.

## Sobre

Esses exercícios foram feitos para praticar estruturas de controle como `if/else` e `switch`, além de operadores lógicos e entrada de dados pelo console.

## Exercícios

### Lista 3 — If/Else básico
- **Ex1:** Verifica se um número é par ou ímpar
- **Ex2:** Verifica se um número é positivo ou negativo

### Lista 4 — Condicionais aplicadas
- **Ex1:** Lê o valor de uma conta de luz e avisa se o gasto foi alto ou normal
- **Ex2:** Compara dois números e exibe a soma (se iguais) ou a diferença (se diferentes)

### Lista 5 — Switch/Case
- **Ex1:** Recebe um número de 1 a 7 e exibe o dia da semana correspondente
- **Ex2:** Recebe o código de um livro e exibe sua categoria

### Lista 6 — Operadores lógicos
- **Ex1:** Lê o nome e preço de três produtos e exibe o mais barato
- **Ex2 (extra):** Simula um semáforo para pedestre ou motorista usando operadores lógicos

## Tecnologias

- C#
- .NET (Console Application)
- Visual Studio

## Como rodar

Cada exercício é um projeto separado. Para executar:

1. Abra a pasta do exercício desejado no Visual Studio
2. Pressione F5 para compilar e rodar
